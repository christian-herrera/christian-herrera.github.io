1)  Instalacion de GIT
      https://git-scm.com/download/win

2)  Inicializo GPG, importo la clave publica del Yubikey
      gpg -k
      gpg --card-status
      gpg --edit-key
        fetch
        q
      gpg --key-edit C19605648A288D24F13285C2992DEE6EC349CA83
      trust
      q
 
3)  Configuro el Socket SSH
      export SSH_AUTH_SOCK=$(gpgconf --list-dirs agent-ssh-socket)

4)  Configuro la Clave GPG a utilizar y las funciones de GIT
      git config --global user.signingkey C19605648A288D24F13285C2992DEE6EC349CA83
      git config --global gpg.program gpg
      git config --global commit.gpgsign true
      
      git config --global user.name "Nombre"
      git config --global user.email "email@dominio.com"

5)  Si esta todo configurado, utilizo:
      git commit -S -m "Comentario"                                           >     Para crear commits firmados
      git remote set-url origin git@github.com:username/your-repository.git   >     Para configurar el url del repositorio remoto utilizando SSH
